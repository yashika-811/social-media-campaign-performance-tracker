# Social Media Campaign Performance Tracker

Track, analyze, and optimize social media ad performance using data-driven insights.

##  Project Overview
This project analyzes social media advertising data to evaluate campaign performance.  
The dashboard helps identify top-performing ads, audience behavior, and ROI to support marketing decisions.

##  Objectives
- Measure ad performance using KPIs like CTR, Clicks, Impressions, and ROI
- Identify high-performing age groups and audience segments
- Analyze interest-based targeting effectiveness
- Support data-driven campaign optimization

## Tools Used
- Power BI (Dashboard & Visualization)
- Power Query (Data Cleaning & Transformation)
- Excel / CSV (Data Source)

## Key Metrics
- Impressions
- Clicks
- CTR (%)
- Spend
- Approved Conversions
- ROI (%)

##  Dashboard Features
- KPI Cards for overall performance
- Gauge visual for CTR tracking
- Audience insights by gender
- Age group vs CTR analysis
- Interest-based performance analysis
- Top 10 performing ads table


## Outcome
This dashboard provides a complete view of social media ad performance and helps marketers optimize future campaigns based on insights.
